<img width="1392" alt="Screenshot 2023-06-23 at 3 49 45 PM" src="./assets/vanna_demo.gif">

# Install

```bash
python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt
```

# Configure

You can configure secrets in `.env` .

```bash
cp .env.example .env
```

# Run

```bash
streamlit run app.py
```
