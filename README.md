# Chatdb.AI  App

# Clone 
```bash
git clone https://github.com/Incrisz/chat-with-database.git
cd chat-with-database
```

# Install
```bash
python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt
```

# Configure
You can configure secrets in `.env` .
```bash
cp .env.example .env
```

# Run
```bash
streamlit run app.py
```
